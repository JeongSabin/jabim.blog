## 🚀 실행 방법

```bash
git clone https://github.com/lee-sihun/sihun.dev.git
cd sihun.dev
npm i
npm run dev
```

## 🛠 기술 스택

- **프레임워크**: Next.js 14 (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **콘텐츠 관리**: ContentLayer
- **댓글 기능**: Giscus
- **배포**: Vercel

## ✨ 주요 기능

- 반응형 디자인 (Tailwind CSS)
- 다크 모드 지원 (next/themes)
- 댓글 기능 (Giscus)
- 검색 엔진 최적화 (SEO)
