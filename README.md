## 🛠 Tech Stack

- **프레임워크**: Next.js 14 (App Router)
- **언어**: TypeScript
- **상태 관리**: Zustand
- **스타일링**: Tailwind CSS
- **콘텐츠 관리**: ContentLayer
- **댓글 기능**: Giscus
- **배포**: Vercel

## ✨ Key features
- SSG(Static Site Generation) 적용 
- 반응형 디자인 (Tailwind CSS)
- 다크 모드 지원 (next/themes)
- 댓글 기능 (Giscus)
- 검색 엔진 최적화 (SEO)
- 카테고리 & 태그 분류
- 게시글 검색
